# Harjoitus 3: Plant Mock API – Kolmannen osapuolen API:n käyttö

Tämä harjoitus opettaa **Plant Mock API** -palvelun käynnistämisen ja käytön Scalar API -dokumentaatiolla.

**→ Täydelliset ohjeet:** [Harjoitus-3-Plant-API.md](Harjoitus-3-Plant-API.md)

## Lyhyt yhteenveto

1. **Osa A** – Käynnistä Plant Mock API ja testaa se **Scalar API -dokumentaatiolla** (`/docs`)
2. **Osa B** – Tutustu verkossa oleviin julkisiin API:hin

Erillistä backend- tai frontend-integraatiota ei vaadita – riittää, että Plant API käynnistyy ja käyttäjä pystyy käyttämään Scalar-dokumentaatiota endpointtien testaamiseen.

## Esivaatimukset

- Python 3.11+
- Docker ja Docker Compose

## Plant Mock API -polku

Plant Mock API -projekti sijaitsee kurssin materiaaleissa. Tyypillinen polku:

- `../Plant-api/plant-mock-api` (suhteessa webdev26-projektiin)
- tai kurssin jakama polku

Jos projektia ei löydy, pyydä opettajalta tai tarkista kurssimateriaalit.
